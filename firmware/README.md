## Features

- **High-Speed Step Generation**: Up to 200kHz+ step rate
- **Hardware Timer-Based**: Deterministic timing
- **Motion Queue Buffer**: 32-64 segments on device
- **Real-Time Status**: 50-100Hz position updates
- **Limit Switch Handling**: Interrupt-driven safety
- **Spindle Control**: PWM or 0-10V analog
- **Coolant Control**: GPIO outputs

## Common Protocol

All platforms implement the same communication protocol:
- Packet-based framing with CRC-16
- See `../comm/` for protocol specification
- USB CDC or UART interface
